# Motivação

## Efeito de Aparição das foto durante scrow

- O efeito que mais gostei desse site é o efeito de aumentar o zoon das fotos a medida que o scrow acontece.
- O efeito reverso também acontece, voltando as fotos pra dentro de um book imaginário.


# Ideia

- Colocar esse mesmo efeito das fotos em uma página de portifólio.
- Porem, ao invés de usar fotos, usar DIV com a imagem do projeto ou do que será mostrado como portifólio
